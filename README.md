## Instalação

- `git clone https://github.com/brenofpsilva/backend.git`
- `cd backend/`
- `composer install`
- `cp .env.example .env`
- Atualize `.env` e defina suas credenciais de banco de dados 
- `php artisan key:generate`
- `php artisan migrate`
- `npm install`
- `npm run dev`
- `php artisan serve`
